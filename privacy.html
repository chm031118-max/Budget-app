Privacy Policy

The Personal Budget App is a client-side budgeting tool created for coursework demonstration purposes. The app does not require account registration and does not send personal budget data to a server.

Data We Store

Browser localStorage is used to store budget entries, language preference, and cookie consent status.

How We Use Data

Stored data is used only to display budget information inside the browser and remember user preferences after refreshing the page.

Cookies and Local Storage

The app does not use third-party tracking cookies, advertising cookies, analytics, or profiling. It uses localStorage only for app data and consent-related preferences.

User Control

Users can clear stored data at any time through their browser settings by deleting localStorage for this site.
